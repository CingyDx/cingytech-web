About

WorldGuess is a fan-made geography game.

It uses original UI, generated SVG assets and mock locations. It does not use official GeoGuessr branding, name, logo or assets.

Tech

HTML5, CSS3, vanilla JavaScript, localStorage, Netlify Functions, Netlify Blobs and an optional Google Maps / Street View browser key.

Set `GOOGLE_MAPS_BROWSER_KEY` in Netlify environment variables to enable the future Google Maps integration path.
