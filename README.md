# Cingy.Tech web

Statický prezentační web pro Cingy.Tech.

## Zaměření

- tvorba webových stránek pro lokální firmy a živnostníky
- reference a ukázky hotových webů
- kontaktní formulář přes Netlify Forms
- sekundární IT služby a servis

## Deploy

Netlify publikuje adresář `public/`. Není potřeba build krok.

Po pushi do větve `main` Netlify automaticky spustí produkční deploy.
